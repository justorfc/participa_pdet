# Directorio de Datos Procesados

Este directorio está destinado a almacenar resultados intermedios y procesados del análisis de entrevistas PDET.

## Contenido esperado

- **Codificaciones**: Archivos CSV con códigos asignados a fragmentos de texto
- **Análisis estadísticos**: Resultados de análisis cuantitativos, frecuencias, coocurrencias
- **Datos procesados**: Cualquier dato transformado o procesado durante el flujo de trabajo

## Nota

Los archivos en este directorio no se incluyen en el control de versiones (excepto este README) para mantener el repositorio limpio. Los resultados se pueden regenerar ejecutando la aplicación Streamlit.
